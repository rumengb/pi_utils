Utility scripts for PixInsight
==============================
Author: Rumen Bogdanovski
Date: 2024-12-26

INSTALLATION
------------
1. Copy *.js to "<PixInsight>/src/scrits/misc" where <PixInsight> is PixInsight base installation folder.
2. Open PixInsight and navigate to "SCRIPT -> Feature Scripts..."
3. In Feature Scripts dialog click add and navigate to "<PixInsight>/src/scrits/misc" and click "Choose"
4. You should see a message saying that new scripts are found and cklick "Ok".
5. The scripts will show under "SCRIPT -> Image Processing"
